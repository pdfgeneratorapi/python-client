# FormatParam

Document format. The zip option will return a ZIP file with PDF files.

## Enum

* `PDF` (value: `'pdf'`)

* `HTML` (value: `'html'`)

* `ZIP` (value: `'zip'`)

* `XLSX` (value: `'xlsx'`)
