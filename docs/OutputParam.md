# OutputParam

Response format. When the \"file\" option is used the API returns the file inline. With the \"url\" option, the document is stored for 30 days and automatically deleted.

## Enum

* `BASE64` (value: `'base64'`)

* `URL` (value: `'url'`)

* `FILE` (value: `'file'`)
